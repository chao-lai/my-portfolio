<ProjectCard title="Send Chinatown Love" link="https://github.com/sendchinatownlove" bg="linear-gradient(90deg, rgba(194,102,142,1) 18%, rgba(35,88,151,1) 100%)">
This open source project is an online fundraising platform built in TypeScript, React, Ruby on Rails. 
Mission: to support cash-only, small businesses severely impacted by COVID-19 (100% of all donations goes straight to the merchant); to bring back foot traffic to the Chinatown neighborhood
Impact: $60,353 raised in donation to 13 local merchants; 6997 vouchers purchased
My contribution involves frontend development using React in TypeScript.
</ProjectCard>

<ProjectCard title="Rabbit" link="https://github.com/chao-lai/rabbit" bg="linear-gradient(to left bottom, #274d86, #005883, #005f70, #006154, #2b6139)">
A mini-Reddit clone built using React in Next.js, Typescript, Node, PostgresQL, GraphQL. Features: login/sign up, fetch a list of posts, fetch a post by ID, create/update/delete a post.	Designed and built a GraphQL schema to define the type of data to be read and written. Used mutation hooks & Apollo client to make requests to the GraphQL server and manage local state.
</ProjectCard>
